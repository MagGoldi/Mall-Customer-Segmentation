# 🛍️ Кластеризация покупателей торгового центра (Mall Customer Segmentation)

## 📌 Задача
**Задача:** Кластеризация клиентской базы на основе их социально-демографических характеристик и покупательского поведения.

В данном проекте проводится анализ данных о покупателях торгового центра для выявления однородных групп (сегментов). Это позволяет бизнесу лучше понимать свою аудиторию и разрабатывать персонализированные маркетинговые стратегии.

## 🔗 Датасет
Данные взяты с платформы Kaggle:
[Mall Customer Segmentation Data](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)

## 🧪 Алгоритмы и основные эвристики
В ходе исследования были применены и сравнены несколько методов обучения без учителя:

1.  **Предобработка данных:**
    - Использование `LabelEncoder` для преобразования категориального признака Gender.
    - Стандартизация данных с помощью `StandardScaler` для приведения характеристик к единому масштабу.
2.  **K-Means (K-средних):**
    - Основной алгоритм для сегментации. 
    - **Метод «Локтя» (Elbow Method):** Использование `WCSS` (Within-Cluster Sum of Squares) для предсказания оптимального числа кластеров.
    - **Силуэтный анализ (Silhouette Score):** Оценка качества разделения и компактности кластеров.
    - Оптимизация параметров проводилась с помощью `GridSearchCV`.
3.  **Agglomerative Clustering:**
    - Иерархический подход с построением **дендрограмм** для визуализации структуры вложенности потребительских групп.
4.  **DBSCAN / HDBSCAN:**
    - Использовались для выявления аномалий (выбросов) и поиска кластеров произвольной формы, не зависящих от жесткого задания количества групп.
